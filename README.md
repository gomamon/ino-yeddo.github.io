# 결혼식 청첩장

모바일 청첩장 GitHub Pages

## 로컬 테스트

### 방법 1: npm 사용
```bash
npm install
npm start
```
브라우저에서 `http://localhost:3000` 접속

### 방법 2: Python 사용
```bash
python3 -m http.server 3000
```
브라우저에서 `http://localhost:3000` 접속

### 방법 3: Node.js http-server 직접 사용
```bash
npx http-server -p 3000
```
브라우저에서 `http://localhost:3000` 접속

## 배포

GitHub Pages에 푸시하면 자동으로 배포됩니다.
